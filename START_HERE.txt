                    AoE4 Bot - Where to Start
================================================================================

IMPORTANT: You're currently in WSL, but this project needs to run on Windows
to interact with Age of Empires IV.

NEXT STEPS:
================================================================================

1. Access this project from Windows File Explorer:
   
   Open File Explorer and go to:
   \\wsl$\Ubuntu\home\mattc\aoe4bot

2. Copy the folder to Windows (recommended):
   
   Suggested location:
   C:\Users\YourName\Projects\aoe4bot

3. Install Python on Windows:
   
   Download from: https://www.python.org/downloads/
   CHECK "Add Python to PATH" during installation!

4. Open PowerShell in the project folder:
   
   Right-click in the folder → "Open PowerShell window here"

5. Run the setup script:
   
   .\setup_windows.bat

6. Test screen capture:
   
   .\run_test.bat
